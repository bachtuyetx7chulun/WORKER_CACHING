# Worker-core

Worker is make by express, bull, redis, there're simple but high performance

## Installation

```js
use this template
```
## Usage

```bash

# Install dependencies
yarn install

# Using dev to develop
yarn dev

# Using project to product
yarn build -> yarn start

```
